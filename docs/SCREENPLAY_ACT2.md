# 《雾会记得》剧本 - 第二幕

## 第二幕：铁锈医院与肉墙地下道 - 记忆的追击

### 场景 7: 铁锈医院 - 聆听者的审判

**分镜**:
1. **建立镜头**：废弃医院外观，锈红色的墙面，破碎的窗户上残留着"急诊"的标识
2. **内景 - 走廊**：病床推车倾倒，医疗器械散落，地面有干涸的血迹
3. **主观镜头**：沈迴的手电光扫过墙上的标识牌："地下停尸房 ↓"
4. **特写**：沈迴的脚步停顿，呼吸变重

**对白**:
```
沈迴（自言自语）：
"我不该下去……"

（远处传来金属拖行声）

沈迴（咬牙）：
"但她在下面。"
```

**心理刻画**:  
沈迴知道地下室是"那个地方"——事故发生的核心现场。他的恐惧不是来自怪物，而是来自**即将面对的真相**。

### 场景 8: 第一次遇见聆听者

**分镜**:
1. **黑暗中的声音**：画面几乎全黑，只有沈迴的手电光锥
2. **音效特写**：一个呼吸声，非常近，但看不见任何东西
3. **慢镜头**：沈迴转身，手电光扫过一个贴着墙壁的身影——聆听者
   - 外形：头部畸形，耳朵异常巨大，眼睛只是两个凹陷的黑洞
   - 姿态：静止不动，但"脸"朝向沈迴
4. **沈迴的反应**：本能想跑，但脚步声让聆听者立刻转头
5. **教学时刻**：画面文字提示："它看不见，但能听见一切。站定不动。"

**互动设计**:
- 玩家必须松开所有移动键
- 屏幕边缘出现"噪音指示器"，显示玩家制造的声音
- 如果玩家移动，聆听者会瞬间冲向玩家

**对白**:
```
沈迴（憋住呼吸，心中独白）：
"别动……别出声……"

（聆听者缓慢移动，从沈迴身边经过）

沈迴（颤抖）：
"它在听什么？听我的心跳？还是……我说不出口的话？"
```

**象征意义**:  
聆听者代表**沉默的审判**。它不需要看见你，因为它已经听到了你内心的声音——那些你说不出口的忏悔和辩解。唯一的对策是**彻底停止**，象征"停止辩解，停止逃避"。

### 场景 9: 发现第三个证物 - 改写的事故报告

**分镜**:
1. **场景**：医院地下档案室，文件柜倾倒，纸张散落一地
2. **特写**：沈迴捡起两份文件，并排放在地上
   - 左边：原始事故报告（手写，多处涂改）
   - 右边：正式事故报告（打字版，干净整洁）

**证物内容对比**:

**原始版本（手写）**:
```
事故时间：2019年9月15日 23:47
地点：灰洄镇疗养院地下蓄水层
伤亡人数：3人重伤，1人失踪
责任方：疗养院违规改建，导致地基渗水
处理医生：沈迴
备注：（手写）"水位上涨过快，只能先救一个。林晨伤势最重但可控，
另外两人在更深处，尝试会导致三人全亡。我选择了确定性。"
```

**正式版本（打字）**:
```
事故时间：2019年9月15日
地点：灰洄镇疗养院
伤亡人数：1人死亡（林晨，抢救无效）
责任方：自然灾害，不可抗力
处理医生：沈迴
结论：医护人员已尽全力，无责任追究。
```

**对白**:
```
沈迴（声音颤抖）：
"他们把三个人改成了一个……
把'我选择了'改成了'抢救无效'……
把真相改成了……"

（手电光闪烁）

沈迴（愤怒）：
"他们让我成了唯一记得的人！"
```

**心理刻画**:  
这是**愤怒阶段**的爆发。沈迴意识到，真相被系统性地抹除了，而他作为当事人，成了唯一的"证人"和"帮凶"。他的愤怒既指向掩盖真相的人，也指向默许这一切的自己。

**分镜细节**:
- 特写：沈迴握紧两份报告，手背青筋暴起
- 闪回插入（5秒）：
  - 一间办公室，一个穿西装的男人（顾承）把正式报告推到沈迴面前
  - 顾承的声音："签字，一切就结束了。你也是受害者。"
  - 沈迴的手拿起笔，停顿，最终签下名字
- 回到现实：沈迴把两份报告都收进口袋

### 场景 10: 遇见老吴 - 第一个同行者

**分镜**:
1. **远景**：医院地下室深处，雾中出现一个人影，背靠墙壁蹲着
2. **接近镜头**：沈迴举起手电，看清那是一个穿着旧消防外套的老人
3. **正反打**：
   - 老吴抬头，眼神疲惫但清醒："你也来找他们了？"
   - 沈迴警惕："你是谁？"
   - 老吴："灰洄镇消防队，退休……或者说，被退休了。"

**对白**:
```
老吴（苦笑）：
"那天晚上，我在现场。我听到了呼救声，
但我没进去。我告诉自己'水位太高了'，
但真相是……我怕死。"

沈迴（沉默，然后）：
"所以你也在这里。"

老吴（站起身，拍拍外套）：
"雾不放过任何人。但也许……我们能一起走出去。"
```

**互动选择**（第一次道德抉择）:
- 选项A："跟我来。"（救下老吴，增加同行者）
- 选项B："我自己走。"（拒绝老吴，孤身前进）

**心理意义**:  
老吴代表**旁观者的罪责**。他没有直接造成伤害，但他的不作为也是真相被埋葬的一部分。沈迴选择是否救他，本质上是在问：**那些"没做错但也没做对"的人，值得被原谅吗？**

**分镜**（如果选择救下）:
- 老吴跟在沈迴身后，保持一定距离
- 偶尔的对话打破寂静，但两人都不愿深谈
- 画面左下角出现："同行者：老吴 信任度：20%"

### 场景 11: 肉墙地下道 - 水纹追踪器

**分镜**:
1. **建立镜头**：一条狭窄的地下通道，墙壁上覆盖着肉质纹理（象征被压抑的罪恶感）
2. **音效设计**：墙壁在"呼吸"，发出低沉的脉动声
3. **特写**：地面积水中出现发光的圆盘——水纹追踪器

**拾取道具后的效果**:
- 画面上开始显示怪物移动留下的虚线路径（8个残影点）
- 每条路径有不同颜色：游荡者（灰）、追猎者（红）、聆听者（蓝）

**对白**:
```
老吴（看到追踪器）：
"这是什么？"

沈迴（捡起，观察）：
"它能看到……轨迹。它们留下的痕迹。"

老吴（低声）：
"就像我们留下的痕迹？"

沈迴（沉默片刻）：
"……对。"
```

**象征意义**:  
水纹追踪器让玩家"看见路径"，象征**回溯记忆**的能力。我们无法改变过去，但可以理解它是如何形成的。

### 场景 12: 第一次遇见追猎者 - 顾承的具象化

**分镜**:
1. **远处的脚步声**：沉重、规律，像军靴踩在水中
2. **雾中的轮廓**：一个高大的身影，穿着破碎的西装，面部扭曲
3. **特写**：追猎者的"脸"——顾承的五官，但眼睛是空洞，嘴角撕裂到耳根
4. **它开口说话**（扭曲的声音）：
   ```
   "沈医生……签字了吗？
   签字就没事了……
   签字……签字……"
   ```

**对白**:
```
沈迴（后退，声音颤抖）：
"你已经死了……"

追猎者（继续逼近）：
"但你还记得……
记得……记得……"

老吴（拉住沈迴）：
"跑！它不会停的！"
```

**追逐场景**:
- 镜头：手持跟拍，剧烈晃动
- 音效：追猎者的脚步声始终在身后，不论距离多远
- 机制：即使玩家躲进房间，追猎者仍会在门外徘徊很久（"死咬"特性）

**心理意义**:  
追猎者代表**迟来的追责**。顾承本人已死，但他强加给沈迴的罪责感仍在追赶。它的台词"签字了吗"反复出现，象征沈迴无法摆脱的**共犯感**。

**分镜细节**:
- 逃跑过程中，闪回插入（1秒）：顾承在办公室里的笑容
- 追猎者的声音与顾承的声音重叠："你也是受害者。"
- 沈迴的内心独白："如果我也是受害者，为什么我会在这里？"

### 场景 13: 发现第四个证物 - 林晨的遗书

**分镜**:
1. **场景**：地下道尽头的一个小房间，像是临时避难所
2. **房间布置**：一张折叠床，一盏手提灯（已熄灭），墙上贴着医疗记录
3. **特写**：沈迴在床上发现一个防水袋，里面是一封信

**遗书内容**:
```
沈医生：

如果你看到这封信，说明我没能走出去。

我不怪你。真的。

那天晚上，我看到你的眼神——那是在做一个不可能的选择。
救我，还是去救那两个人。
你选了我，因为我伤得最重，也因为你知道能救活我。

但我想告诉你：
我宁愿你去试试救他们。

不是因为我不想活，而是因为……
我知道你会一辈子记得这个选择。
我知道你会问自己"如果我去了，是不是能救下所有人"。
我知道你会把"尽力了"改写成"没尽全力"。

沈医生，如果雾真的会记得，
那请记得这件事：
你没有做错。
你只是……也是人。

别再逃了。
去找到他们的名字。
去说出真相。

然后，
请原谅你自己。

林晨
2019年9月16日凌晨
```

**对白**:
```
沈迴（读完信，双手颤抖）：
"原谅自己……"

（长时间沉默）

沈迴（声音破碎）：
"我做不到……林晨，我做不到……"

（老吴走过来，沉默地拍了拍他的肩膀）
```

**心理刻画**:  
这是整个剧本的**情感核心**。林晨的遗书揭示了两个真相：
1. **他人的宽恕**已经存在（林晨不怪他）
2. **自我宽恕**才是真正的难关

沈迴的悲剧不在于他做错了，而在于他**无法接受自己在不可能的情况下做出的选择**。

**分镜细节**:
- 特写：信纸在沈迴手中颤抖
- 泪水滴在信纸上，墨迹晕开
- 老吴的脸：他理解这种痛苦，因为他也在经历

### 场景 14: 第二幕结束 - 再次下沉

**分镜**:
1. **仪式点**：地下道深处出现发光的召回印记
2. **老吴站在印记外**："你先下去。我……还不够格。"
3. **沈迴**："你会够格的。等我回来。"
4. **下沉动画**：建筑再次折叠，雾中传来童声

**过场文字**:
```
"你开始记起他们的名字。
但名字还不够。
你必须归还他们。"

第三层 - 灰烬教堂
```

**心理状态转变**:  
第二幕结束时，沈迴从**愤怒**进入**抑郁与接受**的边缘。他意识到真相不会让痛苦消失，但至少……他不再独自背负。
