# 关卡与玩法深度设计 — The Fog Remembers (Godot 重制)

> 关联：`GODOT_REMAKE_DESIGN.md`（主文档）、`ART_PROMPTS.md`（生图）
> 本文解决原作最大短板：**8 层玩法相同**。这里给每章一个**真正改变操作方式**的核心机制。

---

## 1. 设计原则：机制即叙事

原作的符号系统非常好（雾=回避的记忆、声呐=求证的代价、灰烬=失去者的名字、晨光=停止辩解的出口）。重制的关键是**让玩家通过"怎么玩"体会到这些符号**，而不是只读旁白。

| 符号 | 原作 | 重制（机制化） |
|------|------|----------------|
| 雾 = 被回避的记忆 | 视觉遮挡 | 推进剧情会"驱散局部雾"，但也暴露你给怪物 |
| 声呐 = 求证的代价 | 点亮地图+惊动怪物 | 多章把"被听见"做成主威胁 |
| 灰烬 = 名字 | 旁白 | 第5章"名字归还"解谜 |
| 晨光 = 出口 | 出口发绿 | 第8章"放下"机制：不能战斗，只能等 |

## 2. 八章差异化机制

> `MechanicType` 枚举：`STEALTH_FOG / SOUND_LOCATE / CHASE_PRESSURE / NOISE_CARRY / NAME_RETURN / WATER_LEVEL / MIRROR_RECONSTRUCT / LET_GO`

### 第1章 · 归潮街口（情绪：压抑 / 机制：STEALTH_FOG 雾中潜行）
- **教学层**。基础：移动、手电、捡证物、做仪式、躲游荡者。
- 新增：**"回头惩罚"** —— 呼应线索"雾散之前，别回头看"。若玩家长时间背对前进方向（手电朝身后），雾会缓慢加浓、理智微降。轻量教学，暗示这个世界的规则。
- 怪物：1–2 游荡者。压力 0.72。

### 第1章续 · 废弃公寓（机制：SOUND_LOCATE 声源定位）★核心机制1
- **真实现"声源定位"**：地图里有 4 台收音机（线索），但**只有正在响的那台在小地图上有声波圈**。玩家要靠**音量随距离变化**（Godot `AudioStreamPlayer2D` 的衰减）听声辨位，走到跟前关掉它。
- 每关掉一台，**走廊重新排列**（部分墙体变化 → 移植原作"关闭噪声后走廊重排"的文本，变成真机制）。
- 张力：收音机本身是噪声源，附近的怪物更易被惊动；关掉它既是推进也是降噪。
- 怪物：游荡者 + 引入 stalker。压力 0.8。

### 第2章 · 铁锈医院（机制：CHASE_PRESSURE 狂暴追逐）★核心机制2
- **追逐压迫**：拉急救电闸（线索）会**短暂全图亮起 + 强制附近怪物进入 chase**（移植 `forceChase`，但做成机制核心而非副作用）。
- 玩家必须在"必须拉闸推进"与"拉闸=被追"之间冲刺走位，利用十字架/药水脱身。
- **节奏机制**：心电仪音效随接近出口越来越急促（`AudioDirector` 动态），作为听觉引导。
- 怪物：stalker + 引入 listener。压力 1.35（全局最高之一）。

### 第2章续 · 肉墙地下道（机制：NOISE_CARRY 噪音搬运）★核心机制3
- **噪音搬运**：心脏保险丝（线索）要从拾取点**搬到配电箱（仪式点）**。**越接近配电箱，保险丝跳动越响（噪声值越高）** → listener 越易锁定你。
- 玩家要规划路线：绕远（安全但耗电耗时）vs 直插（快但吵）。
- 可"暂时放下"保险丝降噪躲避，再回来拿 —— 引入轻量物品携带状态。
- 怪物：listener 为主（沉默审判）。压力 1.45（峰值）。

### 第3章 · 灰烬教堂（机制：NAME_RETURN 名字归还）★核心机制4
- **名字归还解谜**：场景里有 N 个跪着的**无脸幻影**（仪式点）和 N 个散落的**名字残页**（线索，每张写不同名字/碎句）。
- 玩家要**把正确的名字交给正确的幻影**（靠线索文本里的线索匹配，例如"生日歌唱错了"对应某个幻影的姿态）。
- 匹配对：幻影起身、哭声变合唱（正反馈）；匹配错：短暂惊动 + 理智小降。
- **节奏放慢**：怪物压力骤降（0.55），从"逃命"转向"哀悼"，情绪转折点。
- ★道德钩子：可以选择**不归还、直接抢路逃走** → 影响 guilt 与结局。

### 第3章续 · 沉没学校（机制：WATER_LEVEL 重力与水位）★核心机制5
- **动态水位**：水位周期性涨落。**高水位时部分通道淹没不可通行**（但能游过/漂浮课桌当踏板），**低水位时露出新路径与线索**。
- 玩家要**踩水位节奏**移动：水声会放大噪声（涉水=吵）。
- 钢琴音准作为听觉引导（移植原作设定）。
- 压力 0.45。情绪：悲伤。

### 第4章 · 破晓海岸（机制：MIRROR_RECONSTRUCT 光影重构）★核心机制6
- **光影重构**：场景有可旋转的**反光镜**（仪式点）。玩家旋转镜子，把晨光**折射到墙上的影子**，影子拼出事故真相画面（线索）。
- 解谜：调整多面镜子角度，让光路串起来照亮终点（轻量光路解谜）。
- ★关键叙事：影像逼主角承认"不是我的错"是自我辩解。压力 0.35。

### 第4章续 · 晨曦病房（机制：LET_GO 放下与等待）★核心机制7 / 终章
- **反玩法机制**：这一层**没有怪物，不能战斗，HUD 逐渐消失**。
- 玩家要**主动"放下"道具**（在床头柜逐一交出药水/十字架/手电/婚戒）——每放下一件，触发一段独白。
- 最后走到窗边**静止不动**，晨光自己变宽 → 通关。**任何按键/移动都会让晨光退缩**（呼应"静止，呼吸，等它经过你"）。
- 这是把"释怀"做成机制的高光设计。压力 0。

### 第5章+ · 无尽下沉（机制：随机抽取上述机制 + 叠加压力）
- 移植 `levelForDepth` 的程序生成：随机组合机制 + 提升怪物压力/速度（`monsterSpeedScale`）。
- 补给更慷慨（`endlessBonus`）但 guilt 持续累积。

## 4. ★ 道德 / 罪责系统（Guilt）与多结局

### 4.1 guilt 变量（隐藏，0–100）
影响事件：

| 行为 | guilt 变化 | 备注 |
|------|-----------|------|
| 救下同伴并带到下一层 | −8 | 直面责任 |
| 让同伴牺牲挡刀（被动触发） | +10 | 把人留在雾里 |
| 主动抛弃同伴（不救/抢路） | +15 | |
| 第3章归还所有名字 | −12 | 哀悼 |
| 第3章拒绝归还、抢路逃走 | +12 | 继续辩解 |
| 第4章让影子承认真相 | −10 | 停止辩解 |
| 第8章完成"放下"全部道具 | −20 | 释怀 |
| 每次声呐（求证）| −1（微） | 愿意看见 |

### 4.2 四结局（在通关第8章 或 主动上岸时结算）

| 结局 | 条件 | 基调 |
|------|------|------|
| **宽恕（晨光）** | guilt 低 + 完成放下 | 主角想起而非遗忘，走出雾 |
| **直面** | guilt 中 + 救下同伴 | 没有原谅也没诅咒，但承认了 |
| **遗忘** | 中途上岸、未完成核心仪式 | 雾暂时退去，但会再来 |
| **沉沦** | guilt 高 + 多次牺牲同伴 | 主角成为雾的一部分（无尽层底） |

### 4.3 "还能走多深"决策点
每完成一章交界，弹出选择：
- **带着证物上岸** → 用当前 guilt 结算一个结局（短局，鼓励重玩）。
- **继续下沉** → 进入下一章，补给更多、怪更凶，guilt 影响延续。

> 这把无尽 roguelite 的"刷"变成"赌"——每深一层都是对"我还敢不敢看下去"的下注。
