# 《雾会记得》剧本 - 最终章：五种结局

> 结局由玩家的累积选择决定，对应游戏 `endings.ts` 中的五种结局类型。
> 决定因素：灰烬归还数、同行者拯救数、是否原谅顾承、三个最终问题的回答、直面真相次数、沉默次数。

---

## 结局一：赎罪·晨光 (redemption)

**触发条件**: 归还所有灰烬 + 救下所有同行者 + 直面真相 + 选择"原谅自己" + 选择"说出真相"

### 场景 E1-A: 走出迷雾

**分镜**:
1. **纯白渐隐**：画面从白光中缓缓显形
2. **真实世界**：沈迴站在灰洄镇疗养院的废墟外，**真实的晨曦**洒在脸上
3. **关键变化**：这一次，没有雾。天空是清澈的蓝灰色，远处海平线上太阳正在升起
4. **特写**：沈迴抬头，闭眼，让阳光照在脸上

**对白**:
```
沈迴（旁白，平静而释然）：
"雾散了。
不是因为它消失了，
而是因为……我不再需要它来隐藏什么。

我记得他们所有人。
我归还了他们的名字。
我会把真相说出去——
即使没有人相信。

因为有些事，
不是为了被相信才去做的。
是为了……让他们存在过。"
```

### 场景 E1-B: 回到人间

**分镜**:
1. **时间跳跃**：字幕"三个月后"
2. **场景**：一间小小的纪念馆 / 社区中心
3. **墙上**：挂着事故遇难者的照片和名字（王建国、陈志远、林晨……）
4. **沈迴**：站在一群人面前，包括记者、遇难者家属
5. **特写**：小雨的母亲（一位中年女性）含泪握着沈迴的手

**对白**:
```
小雨母亲：
"谢谢你……谢谢你让小雨能够……安息。
这些年，我一直不知道她去了哪里……
现在我知道了，她是去找她爸爸了。"

沈迴（温和）：
"他们现在……在一起了。"

（一位记者举手）
记者：
"沈医生，您知道公开这些会让您面临法律风险吗？
那份被改写的报告，您也签了字……"

沈迴（平静）：
"我知道。
我准备好承担后果了。
逃避了七年，
现在……我想堂堂正正地活着。"
```

### 场景 E1-C: 尾声

**分镜**:
1. **海边**：沈迴独自站在破晓海岸（真实的，没有雾）
2. **他从口袋里拿出那个漂流瓶**（写给自己的信）
3. **特写**：他把信重新读了一遍，然后微笑
4. **他没有扔掉瓶子，而是收好**
5. **远景**：阳光下，沈迴转身离开，背影坚定

**最终独白**:
```
沈迴（旁白）：
"林晨说，请原谅自己。

我花了七年才明白：
原谅自己，
不是忘记那个晚上，
而是带着它，继续活下去。

雾会记得。
我也会记得。
但记得，不再是惩罚——
而是……承诺。

我答应过要记住你们。
我做到了。
现在，
也请你们……看着我，好好活着。"
```

**画面定格**:
- 阳光、海浪、沈迴远去的背影
- 字幕浮现："雾会记得每一个被记住的名字。"
- 完

**情感落点**: **救赎与希望**。沈迴完成了从逃避到承担的完整旅程，获得真正的内心平静。

---

## 结局二：真相·沉默 (truth)

**触发条件**: 找到所有证物 + 直面真相 + 选择"原谅自己" + 但选择"保持沉默"

### 场景 E2-A: 走出迷雾，但独自背负

**分镜**:
1. **白光渐隐**：沈迴走出废墟，晨曦微弱
2. **天空**：雾大部分散去，但地平线上还残留着一层薄雾
3. **特写**：沈迴握着所有证物（报告、遗书、日志），站在原地良久
4. **慢动作**：他将证物一份份收进口袋——没有拿出来公开

**对白**:
```
沈迴（旁白）：
"我知道了真相。
我归还了名字。
我原谅了自己。

但有些真相……
说出来只会带来更多痛苦。

小雨的母亲，
陈志远的未婚妻，
他们已经用七年时间，
学会了与'意外'共存。

如果我现在说出'其实有人本可以被救'，
说出'报告被改写了'，
那只会重新撕开他们的伤口。

也许……
有些真相，
只需要一个人记得，就够了。"
```

### 场景 E2-B: 沉默的守护者

**分镜**:
1. **字幕**："此后每年9月15日"
2. **蒙太奇**：
   - 沈迴独自来到海岸
   - 在沙滩上摆放四盏小灯（代表四位遇难者）
   - 点燃，看着它们在海风中摇曳
   - 默默坐到天亮
3. **特写**：他的脸——平静，但带着永恒的孤独

**对白**:
```
沈迴（对着大海，轻声）：
"王建国，陈志远，林晨……
还有那个晚上之后走失的小雨……

我记得你们。
每一年，我都会来。
即使全世界都忘了，
即使没有人知道真相，
我会替你们记得。

这是我选择的赎罪方式——
不是喧哗的正义，
而是沉默的守候。"
```

### 场景 E2-C: 尾声

**分镜**:
1. **多年后**：沈迴老了，头发花白
2. **他依然在每年9月15日来到海岸**
3. **这一次，他带来了一个年轻人**（暗示他收的学生/继承者）
4. **特写**：他把"记住"的责任，悄悄传递下去

**最终独白**:
```
沈迴（旁白，苍老）：
"雾会记得。
我也会记得。
直到我也变成雾的一部分。

到那时，
会有人替我记得吗？

我不知道。
但至少，
我尽力了。

就像那个晚上一样——
我，尽力了。"
```

**画面定格**:
- 两个身影（老沈迴和年轻人）站在海边
- 四盏小灯在晨曦中渐渐熄灭
- 字幕："有些记得，是无声的。"
- 完

**情感落点**: **孤独的救赎**。沈迴获得了内心平静，但选择独自背负真相，是一种悲悯但孤独的结局。

---

## 结局三：牺牲·留守 (sacrifice)

**触发条件**: 救下所有同行者 + 但未归还所有灰烬（或部分名字未归还）+ 选择留在雾中

### 场景 E3-A: 不愿离开

**分镜**:
1. **发光的门前**：沈迴停下脚步，回头看
2. **身后**：老吴和小雨在更深的雾中等待（他们救了，但还有未归还的灰烬）
3. **特写**：沈迴看着手中剩余的灰烬罐（还有几个名字未归还）

**对白**:
```
沈迴（旁白）：
"门就在眼前。
外面是阳光，是重新开始的机会。

但我手里……还有名字没有归还。
还有人……在雾里等着。

如果我现在出去，
他们就永远……留在这里了。"

林晨（轻声）：
"沈迴，门不会一直开着。
你确定吗？"

沈迴（转身，背对门）：
"我确定。
我不能丢下他们。
就像那个晚上……
不，正是因为那个晚上，
这一次，我一个都不能少。"
```

### 场景 E3-B: 永恒的守夜人

**分镜**:
1. **沈迴走回雾中**：发光的门在他身后缓缓关闭
2. **他回到灰烬教堂**：继续一个个归还剩余的名字
3. **蒙太奇**：
   - 沈迴在无尽的雾中行走
   - 寻找散落各处的灰烬
   - 一个个归还，一个个送别
4. **特写**：他的身影逐渐变得半透明（他正在成为雾的一部分）

**对白**:
```
老吴：
"你为我们……放弃了出去的机会。"

沈迴（微笑）：
"不是放弃。是选择。
有些人选择走出去，
有些人选择留下来守护。

我已经逃避了七年。
这一次，
我想留下来，
直到最后一个名字被归还。"

小雨（牵住他的手）：
"那我陪你。"

老吴：
"我也是。"
```

### 场景 E3-C: 尾声

**分镜**:
1. **灰洄镇全景**：雾依然笼罩，但不再压抑
2. **雾中**：三个身影（沈迴、老吴、小雨）一起行走，寻找名字
3. **特写**：他们成为了雾中的"指引者"——帮助其他迷失者找到出口

**最终独白**:
```
沈迴（旁白）：
"我没有走出迷雾。
但我不再害怕它了。

现在，我是雾的一部分。
不是被困的囚徒，
而是守夜的灯。

当有人迷失在这里，
当有人背负着无法言说的愧疚，
我会找到他们，
带他们找到名字，
带他们找到……宽恕。

雾会记得。
而我，
就是雾的记忆。"
```

**画面定格**:
- 雾中，三盏微光缓缓移动
- 远处，又有新的迷失者出现，光向他走去
- 字幕："有人选择离开，有人选择成为光。"
- 完

**情感落点**: **崇高的牺牲**。沈迴放弃了个人解脱，选择留下守护他人，是一种利他主义的悲壮结局。

---

## 结局四：否认·再沉 (denial)

**触发条件**: 拒绝归还灰烬（或归还极少）+ 抛弃同行者 + 选择"继续背负"或逃避真相

### 场景 E4-A: 逃离

**分镜**:
1. **发光的门前**：沈迴急切地走向门，不回头
2. **身后**：未归还的灰烬罐散落一地，同行者的呼喊渐远
3. **特写**：沈迴的脸——逃避、恐惧、不愿面对

**对白**:
```
沈迴（旁白，急促）：
"我不想再看了。
不想再记起了。
那些名字，那些脸，那些呼救声……
够了。

我只想出去。
我只想……忘记这一切。"

林晨（悲伤）：
"沈迴，如果你这样走出去……
你什么都没改变。"

沈迴（捂住耳朵）：
"我不管！我只想活下去！
正常地、不带愧疚地……活下去！"
```

### 场景 E4-B: 虚假的解脱

**分镜**:
1. **沈迴冲出门**：外面是刺眼的阳光
2. **但画面逐渐扭曲**：阳光开始变灰，建筑开始模糊
3. **特写**：沈迴回头，发现门后的"现实世界"，正在重新被雾笼罩
4. **恐怖揭示**：他没有走出迷雾，而是走进了**更深的一层**

**对白**:
```
沈迴（环顾四周，恐慌）：
"不……这不对……
这里应该是外面……"

（熟悉的声音从雾中传来）

追猎者（顾承的声音）：
"欢迎回来，沈医生。
逃避……是走不出去的。
你越逃，
雾就记得越深。"

沈迴（崩溃，跪地）：
"不……放我出去……求你了……"
```

### 场景 E4-C: 尾声

**分镜**:
1. **无尽下沉**：沈迴在一层又一层的雾中奔跑
2. **每一层**：都重复着那个晚上的片段，但越来越扭曲
3. **特写**：他的脸逐渐失去人性，开始向"游荡者"转变
4. **最终**：沈迴成为了雾中的又一个怪物——一个永远逃避、永远迷失的灵魂

**最终独白**:
```
（不再是沈迴的声音，而是雾的声音）

旁白：
"他选择了遗忘。
但雾不会遗忘。

那些没有被归还的名字，
那些被抛弃的同行者，
那些没有说出口的真相——
它们会一直、一直地……
记得。

而他，
将永远在雾中下沉，
寻找一个，
他自己亲手关上的出口。"
```

**画面定格**:
- 雾中，一个佝偻的身影缓慢游荡（已变成游荡者）
- 远处，新的探索者举着手电走来
- 那个曾是沈迴的身影，缓缓转向新的猎物
- 字幕："雾会记得每一次逃避。"
- 完

**情感落点**: **堕落与轮回**。沈迴因逃避而永远迷失，成为雾的一部分。这是最黑暗的结局，也是对"逃避"主题的终极警示。

## 结局五：无尽·下沉 (endless)

**触发条件**: 默认结局 / 未完成任何承诺 / 中途放弃 / 道德状态中性

### 场景 E5-A: 未完成的旅程

**分镜**:
1. **灰暗的雾中**：沈迴站在一个十字路口，四周都是雾
2. **没有门，没有出口**：只有无尽延伸的迷雾
3. **特写**：沈迴的表情——疲惫、迷茫，既没有救赎也没有堕落

**对白**:
```
沈迴（旁白）：
"我还在这里。

我没有归还所有的名字，
也没有完全逃避。
我没有救下所有人，
也没有抛弃他们。

我只是……还在走。
还在寻找。
还在……记得，
和遗忘之间，
徘徊。"
```

### 场景 E5-B: 循环

**分镜**:
1. **蒙太奇**：沈迴在不同的层中穿行
   - 归潮街口、废弃公寓、铁锈医院……
   - 每一层都似曾相识，但又略有不同
2. **特写**：他偶尔捡起一个灰烬，归还一个名字
3. **但更多的名字，仍在等待**

**对白**:
```
林晨（声音，温和）：
"沈迴，你随时可以停下来。
随时可以做出选择。
归还，或者放下。
面对，或者逃避。

雾会等你。
它有的是时间。"

沈迴（疲惫地微笑）：
"我知道。
也许有一天，
我会准备好。
但今天……
让我再走一会儿吧。"
```

### 场景 E5-C: 尾声（开放式）

**分镜**:
1. **沈迴继续在雾中行走**：手电的光在雾中划出一道孤独的弧线
2. **远处**：隐约有其他探索者的光（暗示更多迷失的灵魂）
3. **画面拉远**：整个灰洄镇在雾中缓缓下沉，又缓缓浮起，循环不息

**最终独白**:
```
沈迴（旁白）：
"雾会记得。

它记得我的每一步，
每一次归还，
每一次逃避，
每一次徘徊。

也许有一天，我会走出去。
也许有一天，我会沉下去。
也许有一天，我会成为雾本身。

但今天，
我还在这里。
还握着手电，
还记得一些名字，
还……在寻找。

这就够了。
至少现在，
这就够了。"
```

**画面定格**:
- 雾中，一束手电光缓缓移动
- 字幕："旅程仍在继续。"
- 提示："（继续游戏，做出新的选择，改变你的结局。）"
- 完（循环至无尽模式）

**情感落点**: **悬而未决的徘徊**。这是 roguelite 的核心结局——没有终点，只有不断的选择与重来。象征人生中那些"还没准备好面对"的时刻。

## 五种结局对照表

| 结局 | 核心情感 | 触发关键 | 主题 |
|------|---------|---------|------|
| 赎罪·晨光 | 救赎与希望 | 全部完成 + 说出真相 | 承担带来自由 |
| 真相·沉默 | 孤独的救赎 | 真相 + 原谅自己 + 沉默 | 无声的守护 |
| 牺牲·留守 | 崇高的牺牲 | 救人 + 留下守护 | 利他的选择 |
| 否认·再沉 | 堕落与轮回 | 逃避 + 抛弃 | 逃避的代价 |
| 无尽·下沉 | 悬而未决 | 中性 / 未完成 | 永恒的徘徊 |

## 结局设计哲学

**为什么是这五个结局？**

1. **赎罪·晨光**：理想结局，但不是"完美"——沈迴依然要承担法律后果，救赎是有代价的。

2. **真相·沉默**：探讨"真相是否一定要公开"的道德困境。沉默不一定是逃避，也可能是慈悲。

3. **牺牲·留守**：质疑"个人解脱是否高于他人"。沈迴选择不出去，体现集体主义的价值取向。

4. **否认·再沉**：警示性结局，揭示"逃避"的终极后果——成为自己最恐惧的东西。

5. **无尽·下沉**：元叙事结局，承认人生没有标准答案，有些创伤需要反复面对才能愈合。

**共同的核心**: 每个结局都不评判玩家"对错"，而是呈现选择的**后果与意义**。这正是"雾会记得"的精神——它不审判，只是**记得**。
