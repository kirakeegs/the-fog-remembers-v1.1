# IP Bible: 雾会记得

## 名称

- 中文名：雾会记得
- 英文名：The Fog Remembers
- 类型：原创心理恐怖 roguelite
- 一句话卖点：在会记忆罪责的港镇迷雾中下沉，用声音与光寻找真相，完成仪式并活下去。

## 世界观

灰洄镇是一座被废弃疗养院、港口仓库、坡道公寓和海岸学校切开的旧港镇。多年以前，一场雨夜事故被集体改写成“无法追责的意外”。雾从疗养院地下层涌出后，镇上的地点开始向下折叠，每一层都把玩家带回事故相关的记忆遗址。

雾不是怪物，也不是天气。它是一套会记录逃避、遗忘和自我辩解的环境机制。玩家越深入，雾越慷慨地给出补给，也越频繁地把未处理的记忆变成追击者。

## 主角

暂定名：沈迴。

沈迴曾在灰洄镇参与过一场被遮掩的救援事故。他回到镇上不是为了查明“谁是凶手”，而是为了确认自己是否也把某个人留在了雾里。游戏文本不直接讲明全部秘密，而通过证物、仪式地点、同行者反应和结局逐步拼合。

## 核心符号

- 雾：被回避的记忆，遮挡但也保存方向。
- 手电：玩家愿意看见真相的范围。
- 声呐：主动求证，代价是被听见。
- 收音机杂讯：危险距离与过去声音混在一起。
- 灰烬：失去者的名字，必须被归还。
- 水纹：事故现场的残留路径。
- 晨光：停止辩解后的出口。

## 章节路线

1. 迷雾与原罪：归潮街口、废弃公寓。玩家学习搜证、声源定位和基础逃生。
2. 铁锈与心魔：铁锈医院、肉墙地下道。追逐压迫增强，噪音成为风险资源。
3. 灰烬与挽歌：灰烬教堂、沉没学校。节奏放慢，重点转向归还名字和救同行者。
4. 晨曦与宽恕：破晓海岸、晨曦病房。敌意降低，玩家需要放下补给和等待出口显影。
5. 无尽下沉：完成主线后继续进入重组层，挑战深度和资源管理。

## 怪物图鉴

- 游荡者：标准威胁。象征被忽视的目击者，靠视野和近距离感知追踪。
- 追猎者：长视野、高速度。象征追责迟到后的加倍追赶，丢失目标后仍会追击。
- 聆听者：几乎没有视觉，依赖声音。象征沉默审判，站定和降噪是核心对策。
- 残暴者：重装慢速威胁。象征拒不承认的巨大阻力，需要绕行或定身。
- 匍匐者：低身位高速威胁。象征沿着缝隙扩散的传闻，会利用狭窄通道压迫玩家。

## 道具

- 电池：扩大可见范围并延长探索窗口。
- 药水：短时间护盾。叙事含义是“把死亡延后一次”，不是治愈。
- 十字架：定身附近怪物。叙事含义是临时秩序，不是彻底驱魔。
- 证物：每层真相碎片，驱动出口条件。
- 仪式点：将证物放回正确位置，让出口显影。
