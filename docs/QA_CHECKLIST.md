# THE FOG REMEMBERS QA 检查清单

每次修改标题页、Canvas、HUD、触控、音频、资源路径或玩法主循环后，至少跑一次本清单。

## 自动检查

```bash
npm run build
npm run qa:smoke
```

通过标准：

- 构建成功。
- 首页返回 200。
- 首页 HTML 包含 `THE FOG REMEMBERS` 和 `雾会记得`。
- 标题封面图、召回印记和至少一个标题音轨可读取。

## 桌面手测

环境：浏览器窗口约 `1440x900`。

- 标题页第一屏清楚显示 `雾会记得 / THE FOG REMEMBERS`。
- “坠入迷雾”可点击并进入游戏。
- Canvas 不是空白，主角、雾、墙体和 HUD 可见。
- `WASD` / 方向键可以移动。
- 鼠标可以控制手电方向。
- `Shift` 冲刺会消耗体力并产生风险。
- `Q` 声呐扫描有反馈，并能影响危险感。
- `E` 可以拾取、互动或推进仪式。
- `1` 药水、`2` 十字架在有库存时有反馈。
- `P` 或 `Esc` 可以暂停，暂停菜单可继续、重开、返回标题。
- 死亡或失败后可以重开。

## 移动端手测

环境：浏览器模拟或真机约 `390x844`。

- 标题页文字不溢出，主按钮在首屏或轻微滚动后可见。
- 进入游戏后 Canvas 填满屏幕。
- 触控移动、手电方向和主要按钮可用。
- HUD 不遮挡触控主区域。
- 暂停、返回标题、重开按钮可点。
- 竖屏下没有关键文字被裁掉。

## 视觉检查

保存两张截图作为本轮证据：

- 桌面：`1440x900`
- 移动端：`390x844`

检查重点：

- 画面有明确主体，不是纯黑或纯雾。
- HUD 信息密度可读。
- 标题页不出现外部 IP 名称。
- 怪物、线索、出口或仪式目标至少有一种可感知提示。

## 音频检查

- 标题页 BGM 播放、暂停、切换磁带可用。
- 浏览器首次交互后游戏音频能够解锁。
- 怪物逼近、拾取、扫描或仪式推进有足够辨识度。
- 音量不会盖过重要反馈。

## 发布前检查

- `README.md` 能正确显示中文。
- `public/audio/CREDITS.md` 已记录素材授权。
- 新增资源放入正确目录。
- 新增玩法写入 `docs/IP_BIBLE.md` 或相关设计文档。
- 已记录本轮验证命令和截图结果。
