   - 归还提升理智 +0.08

2. **收音机** (`radio`): 被动探测怪物
   - 自动接收附近怪物杂讯（越近越响）
   - 按 3 主动调频：暴露位置，但短暂标记所有怪物

3. **水纹追踪器** (`rippleTracker`): 显示怪物路径
   - 被动记录每只怪物最近的移动轨迹（8 个采样点）
   - 屏幕渲染虚线路径

4. **手电模式切换** (`flashlightMode`): 广角/聚焦
   - 按 F 切换
   - 广角：视野宽（90°）、距离短（78%）
   - 聚焦：视野窄（33°）、距离远（155%）

**玩法改进**:

### Task #4: 怪物机制差异化 ✅

**每种怪物现在有独立对策**:

1. **聆听者** (Listener)
   - **特性**: 几乎全盲（视野极小），只靠声音
   - **对策**: 站定不动（`playerNoise <= 0.05`）即可隐身
   - **玩法**: 停止移动的生存策略

2. **追猎者** (Stalker)
   - **特性**: 长视野、死咬（丢失目标后仍追击很久）
   - **对策**: 彻底切断视线 + 拉开极大距离
   - **调整**: `alertHold = 4.2`（追击时间更长）

3. **残暴者** (Brute)
   - **特性**: 重装慢速，转向极慢，**正面无法击退**
   - **对策**: 从侧后方绕行（正面攻击无效）
   - **实现**: `attackMonsters` 中检测攻击角度，正面攻击不造成击退

4. **匍匐者** (Crawler)
   - **特性**: 高速贴地，在窄道额外加速 35%，开阔地减速 18%
   - **对策**: 避开狭窄通道，利用开阔地拉扯
   - **实现**: `isInCorridor` 检测地形，动态调整速度

5. **游荡者** (Wanderer)
   - **特性**: 标准威胁，视野+听觉均衡
   - **对策**: 基础逃生技巧

**通用机制**:
- 每种怪物的感知规则完全独立
- 转向速度差异化（残暴者 0.05，追猎者 0.22）
- 不再是数值区别，而是策略差异
