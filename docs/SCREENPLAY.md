# 《雾会记得》长篇剧本

**类型**: 心理恐怖 / 悬疑剧情  
**时长**: 约 120 分钟（适合改编为互动叙事游戏或系列短片）  
**核心主题**: 记忆、逃避、赎罪与宽恕  

---

## 角色表

### 主要角色

**沈迴** (30 岁)  
- 前急救医生，灰洄镇事故当事人之一
- 性格：外表冷静克制，内心充满自责与逃避
- 核心冲突：他知道自己在那个雨夜做了选择，但不确定那是否是"正确"的选择
- 视觉：深色风衣，手持老式手电筒，眼神疲惫但坚定

**林晨** (27 岁，已故)  
- 沈迴的前同事，事故中的牺牲者
- 以幻象、声音和记忆碎片的形式出现
- 性格：温和、理想主义，相信每个生命都值得拯救
- 视觉：护士制服，半透明的雾状轮廓，总在视野边缘

**老吴** (55 岁，同行者)  
- 灰洄镇退休消防员，事故目击者
- 性格：沉默寡言，但有强烈的责任感
- 核心冲突：他本可以说出真相，但选择了沉默
- 视觉：旧消防外套，手里攥着一串生锈的钥匙

**小雨** (8 岁，同行者)  
- 事故中失踪儿童的投影，代表"被遗忘的受害者"
- 性格：安静、不说话，只用眼神交流
- 视觉：脏污的校服，赤脚，手里握着一张褪色的全家福

**疗养院院长·顾承** (65 岁，已故)  
- 事故责任方，试图掩盖真相的核心人物
- 以"追猎者"的形式出现，象征迟来的追责
- 视觉：破碎的西装，面部扭曲，移动时留下水迹

### 怪物化身（象征意义）

- **游荡者**: 被忽视的目击者，那些看见但选择转身的人
- **追猎者**: 顾承的罪责具象化，永不停歇的追赶
- **聆听者**: 沉默的审判者，只有停下来才能面对
- **残暴者**: 拒不承认的心理防御机制
- **匍匐者**: 在缝隙中扩散的流言与传闻

## 结构框架

### 第一幕：归潮街口与废弃公寓（逃避的开始）
- 建立世界观：雾、手电、声呐的机制
- 引入主角动机：沈迴为什么回到灰洄镇
- 第一个证物：急救记录单

### 第二幕：铁锈医院与肉墙地下道（记忆的追击）
- 深化冲突：林晨的幻象首次出现
- 道德选择开始：第一次遇到灰烬（死者名字）
- 第二、三个证物：抢救现场照片、改写的事故报告

### 第三幕：灰烬教堂与沉没学校（归还与挽歌）
- 核心转折：沈迴开始主动归还名字
- 同行者加入：老吴与小雨
- 第四、五个证物：林晨的遗书、小雨的失踪报告

### 第四幕：破晓海岸与晨曦病房（宽恕与选择）
- 高潮：面对追猎者（顾承）的最终对峙
- 关键抉择：救下所有人 vs. 独自逃离
- 最后证物：沈迴自己的急救日志

### 第五幕：结局分支（无尽或晨光）
- 根据玩家选择触发不同结局
- 每个结局都有独立的情感落点

## 第一幕：归潮街口 - 逃避的开始

### 场景 1: 标题序幕

**分镜**:
- 镜头：全黑屏幕，只有呼吸声和远处海浪拍岸的闷响
- 画面逐渐亮起：一只手握着手电筒，光锥在雾中划过
- 俯视镜头：沈迴站在一条被雾吞没的街道中央，身后是折叠向下的建筑轮廓
- 标题浮现：《雾会记得》

**对白**:
```
沈迴（旁白，疲惫）：
"他们说雾会散。
但这里的雾不散，它只是……等着。
等你记起那些你假装忘掉的事。"
```

**心理状态**:  
沈迴此刻处于**麻木与抗拒**之间。他知道自己为什么来这里，但还没准备好直面真相。手电筒是他唯一的控制感来源——"我选择看见什么"。

**音效与氛围**:
- 远处传来收音机杂讯，混杂着断断续续的人声
- 脚步声在湿滑的石板路上回响
- 雾中偶尔传来金属碰撞声（暗示怪物存在）

---

### 场景 2: 归潮街口 - 第一次探索

**分镜**:
1. **建立镜头**（宽景）：破败的港口街道，路灯倾斜，雾从地面缝隙涌出
2. **主观镜头**：沈迴的手电光锥扫过墙面，看到褪色的店招："灰洄镇卫生院 ← 200m"
3. **特写**：沈迴的手微微颤抖，握紧手电筒
4. **中景**：雾中出现一个人形轮廓，缓慢游荡（游荡者首次登场）

**对白**:
```
沈迴（低声自语）：
"还是这条街……什么都没变。"

（手电扫到一面墙上的涂鸦："他们撒谎了"）

沈迴（停顿）：
"……我知道。"
```

**互动提示**（游戏化元素）:
- 玩家学习：WASD 移动，鼠标控制手电方向
- 教学文本："按 F 切换手电模式。广角看得宽，聚焦看得远。"

**发现第一个证物**:
- 地点：街角的废弃急救车旁
- 物品：一张沾血的急救记录单，日期：2019 年 9 月 15 日
- 记录内容：
  ```
  患者编号：#47（林晨）
  伤情：多处撕裂伤，失血性休克
  处理医生：沈迴
  备注：（手写）"我尽力了"
  ```

**心理刻画**:  
沈迴看到记录单时，镜头推近他的脸——眼神闪躲，呼吸急促。他几乎要把证物扔掉，但最终还是收进口袋。这是**第一次逃避**：他拿走了证据，但拒绝细读内容。

**分镜细节**:
- 特写：记录单上的血迹与沈迴指尖重叠
- 闪回插入（0.5 秒）：一双手在急救，但画面模糊、颜色失真
- 回到现实：沈迴闭眼，深呼吸

---

### 场景 3: 第一次遭遇 - 游荡者

**分镜**:
1. **远景**：雾中出现一个佝偻的人形，头部异常扭曲，缓慢转向沈迴
2. **切分屏**：左侧是游荡者的轮廓，右侧是沈迴握紧手电的手（手在发抖）
3. **追逐镜头**：游荡者突然加速，沈迴转身奔跑，手电光锥剧烈晃动
4. **喘息特写**：沈迴躲进一条巷子，背靠墙壁，压抑呼吸声

**对白**:
```
沈迴（喘息，自言自语）：
"不是真的……只是雾……"

（游荡者的脚步声逐渐远去）

沈迴（苦笑）：
"但雾会记得，对吧？"
```

**心理状态**:  
这是**否认阶段**。沈迴试图说服自己这些怪物"不是真的"，但他内心知道——它们是他逃避的记忆具象化。游荡者象征那些"看见但选择转身的人"，包括他自己。

**音效设计**:
- 游荡者的脚步声：湿滑、拖沓，像在水中行走
- 呼吸声混响，营造幽闭恐惧感
- 远处传来微弱的女声："……沈医生……"（林晨的声音）

---

### 场景 4: 废弃公寓 - 收音机与记忆

**分镜**:
1. **建立镜头**：一栋七层公寓楼，窗户破碎，阳台栏杆扭曲
2. **内景 - 一楼大堂**：破碎的信箱，墙上贴满寻人启事（全是模糊的脸）
3. **特写**：沈迴捡起一台老式收音机，按下开关
4. **音效特写**：杂讯中混杂着断续的对话：
   ```
   "……9 月 15 日……疗养院地下……水位上涨……"
   "……来不及了……还有三个人……"
   "……谁去？……"
   ```

**对白**:
```
沈迴（对着收音机，声音颤抖）：
"别说了……我知道了……"

（杂讯突然增强，收音机里传出尖锐的心电图平音）

沈迴（扔掉收音机，捂住耳朵）：
"我说别说了！"
```

**心理刻画**:  
收音机是**强制回忆装置**。沈迴无法控制它播放什么，就像他无法控制记忆涌现。他的反应是愤怒——这是防御机制，用愤怒掩盖恐惧。

**发现第二个证物**:
- 地点：公寓二楼走廊
- 物品：一张抢救现场照片（黑白，边角烧焦）
- 画面内容：
  - 一间地下室，积水到小腿
  - 三个人影：一个躺在担架上（林晨），一个跪在旁边急救（沈迴），一个站在门口（背影模糊）
  - 照片背面手写："为什么只救了一个？"

**分镜细节**:
- 特写：沈迴盯着照片中"站在门口的背影"
- 闪回插入（3 秒）：
  - 画面：积水的地下室，沈迴跪在林晨身边，背后传来呼救声
  - 沈迴的视角：转头看向声音来源，但水位正在上涨
  - 一个选择的瞬间：救林晨（确定能救活），还是冒险去救另一个人（可能两个都救不了）
  - 画面定格在沈迴的脸上——那是**决定的瞬间**
- 回到现实：沈迴双手颤抖，照片掉在地上

---

### 场景 5: 第一次遇见林晨的幻象

**分镜**:
1. **主观镜头**：沈迴继续向公寓深处走，手电光锥扫过走廊尽头
2. **静态镜头**：走廊尽头站着一个女性身影，背对沈迴，穿着护士制服
3. **推镜**：沈迴缓慢靠近，手电光穿透她的身体（半透明）
4. **正反打**：
   - 沈迴的脸：震惊、悲伤、愧疚混合
   - 林晨转身：脸部清晰但眼神空洞，嘴唇微动但没有声音
5. **特写**：林晨举起右手，指向沈迴背后

**对白**:
```
沈迴（声音破碎）：
"林晨……我……"

（林晨摇头，继续指向他身后）

沈迴（转身，看到墙上用血写的字）：
"'他们还在等你。'"

（再转回头，林晨已经消失）

沈迴（跪倒在地，捂住脸）：
"我知道……我一直知道……"
```

**心理刻画**:  
这是**愧疚的具象化**。林晨的幻象不说话，因为沈迴不需要她说——他自己就是最严厉的审判者。她指向的方向象征"未完成的责任"。

**分镜情绪节奏**:
- 前半段：克制、安静，只有脚步声和呼吸
- 林晨出现时：环境音消失，只剩心跳声
- 沈迴跪倒后：远处传来微弱的童声（暗示小雨）

---

### 场景 6: 第一幕结束 - 下沉

**分镜**:
1. **俯视镜头**：沈迴站在公寓楼顶，下方是无尽的雾
2. **特写**：地面出现发光的仪式印记（召回印记）
3. **全景**：沈迴踏入印记，地面开始下沉，建筑折叠

**对白**:
```
沈迴（旁白）：
"雾不会放过我。
但也许……这就是我来这里的原因。
不是为了逃，而是为了……记起。"
```

**过场动画**:
- 画面：公寓楼、街道、雾层层向下折叠
- 文字浮现："第二层 - 铁锈医院"
- 音效：远处传来心电图的滴答声

**心理状态转变**:  
第一幕结束时，沈迴从**否认**进入**愤怒与讨价还价**阶段。他开始承认"雾"（记忆）的存在，但还在试图控制它。
