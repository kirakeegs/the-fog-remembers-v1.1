# 《雾会记得》剧本 - 第三幕

## 第三幕：灰烬教堂与沉没学校 - 归还与挽歌

### 场景 15: 灰烬教堂 - 归还的仪式

**分镜**:
1. **建立镜头**：一座废弃的小教堂，钟楼倾斜，彩绘玻璃破碎
2. **内景**：教堂中央有一个石制祭坛，上面摆放着数十个小陶罐（装着灰烬）
3. **特写**：每个陶罐上都刻着一个名字，但大部分已经模糊不清
4. **远景**：沈迴站在门口，犹豫是否进入

**对白**:
```
沈迴（自言自语）：
"归还名字……但我连他们是谁都不知道。"

（教堂深处传来微弱的童声）

童声：
"你知道的……你一直知道……"
```

**心理状态**:  
沈迴来到了**道德选择的核心**。灰烬教堂是整个游戏中唯一不以"逃生"为目标的关卡——这里没有出口，只有选择：归还还是遗弃。

---

### 场景 16: 第一次拾取灰烬

**分镜**:
1. **主观镜头**：沈迴走向祭坛，手电光照亮第一个陶罐
2. **特写**：陶罐上的名字："王建国，1967-2019"
3. **触摸瞬间**：画面闪烁，出现一段记忆碎片

**记忆碎片（3秒闪回）**:
- 画面：一个中年男人在地下室里呼救，水已经淹到胸口
- 他的声音："救救我……我女儿还在等我……"
- 沈迴的视角：转头看了一眼，然后继续救林晨
- 男人的声音逐渐微弱："……求你了……"

**回到现实**:
```
沈迴（跪倒，捂住脸）：
"我听到了……我听到了他的声音……
但我……"

（长时间沉默）

沈迴（颤抖着拿起陶罐）：
"对不起……对不起……"
```

**互动机制**:
- 玩家按 E 拾取灰烬
- 画面提示："灰烬会记得它的主人。归还它，或者……遗弃它。"
- 拾取后，背包里出现"王建国的灰烬"

**心理意义**:  
拾取灰烬=**承认记忆**。沈迴不再逃避"还有其他人"这个事实，他开始直面自己听到但选择忽略的呼救声。

---

### 场景 17: 遇见小雨 - 第二个同行者

**分镜**:
1. **教堂角落**：雾中出现一个小小的身影，蹲在长椅后面
2. **接近镜头**：沈迴举起手电，看到一个约8岁的女孩，穿着脏污的校服
3. **特写**：小雨抬起头，眼神空洞但清澈，不说话，只是看着沈迴
4. **小雨的手**：紧紧握着一张褪色的全家福（三个人：爸爸妈妈和她）

**对白**:
```
沈迴（蹲下，温和）：
"你叫什么名字？"

（小雨不说话，只是指了指全家福上的字："王小雨，6岁"）

沈迴（震惊）：
"王……王建国的女儿？"

（小雨点头，然后指向沈迴手里的陶罐）

沈迴（理解）：
"你在等他……你一直在等他……"
```

**互动选择**（第二次道德抉择）:
- 选项A："跟我来，我带你去找你爸爸。"（救下小雨）
- 选项B："我帮不了你。"（拒绝小雨）

**如果选择救下**:
- 小雨站起来，牵住沈迴的手（触碰瞬间，画面闪烁）
- 闪回插入：小雨在事故发生前，在疗养院门口等爸爸下班的场景
- 沈迴的内心独白："她不知道爸爸已经……还是说，她知道，只是不愿相信？"

**象征意义**:  
小雨代表**被遗忘的受害者**。她不是事故的直接伤亡者，但她失去了父亲，而真相被掩盖让她连哀悼的机会都没有。她的沉默象征那些"无法发声的人"。

---

### 场景 18: 归还仪式 - 第一个灰烬

**分镜**:
1. **祭坛**：出现发光的仪式印记（与下沉印记不同，这是温暖的金色）
2. **互动提示**："按 G 归还灰烬。这会让雾记得他的名字。"
3. **归还动画**：
   - 沈迴将陶罐放在印记上
   - 陶罐裂开，灰烬飘散成金色光点
   - 光点汇聚成王建国的轮廓（模糊但温和）
   - 他看向小雨，微笑，然后消散
4. **小雨的反应**：第一次哭出声，跪在地上

**对白**:
```
沈迴（轻声）：
"他看到你了。"

小雨（抽泣）：
"……爸爸……"

沈迴（跪下，抱住小雨）：
"对不起……对不起让你等了这么久……"
```

**游戏机制变化**:
- 归还灰烬后，危险度降低 30%
- 沈迴的理智值恢复 8%
- 画面左下角："道德状态 - 灰烬已归还：1"

**心理意义**:  
归还=**承担责任**。沈迴不是在"救"王建国（他已经死了），而是在让他的存在被记住，让他的女儿能够哀悼。这是**超越生死的责任**。

---

### 场景 19: 教堂深处 - 更多的灰烬

**分镜**:
1. **全景**：祭坛后面的墙上，挂着一面巨大的名单
2. **特写**：数十个名字，有些清晰，有些模糊，有些完全消失
3. **沈迴的反应**：倒吸一口凉气

**名单内容**（部分可见）:
```
事故遇难者名单（未公开）

王建国，51岁，疗养院维修工
李梅，34岁，清洁工
陈志远，28岁，护工
……（更多名字）
林晨，27岁，护士
沈迴，30岁，急救医生（手写，被划掉）
```

**对白**:
```
沈迴（颤抖）：
"我的名字……也在上面……"

老吴（走过来，沉默地看着名单）：
"也许……我们都是遇难者。
只是有些人死了，有些人……还活着。"

沈迴（苦笑）：
"活着的，比死了的更痛苦。"
```

**发现第五个证物**:
- 物品：小雨的失踪报告
- 内容：
  ```
  失踪人口：王小雨，6岁
  失踪时间：2019年9月16日
  失踪地点：灰洄镇疗养院门口
  备注：疑似在等待父亲时走失。父亲王建国于前一日在疗养院事故中死亡，
        但官方报告未公开此信息。孩子可能不知道父亲已……
  案件状态：因证据不足，结案。
  ```

**心理刻画**:  
沈迴意识到，这场事故的受害者不仅是死者，还有那些被留下的人——包括小雨，包括死者的家人，也包括他自己。"活着的遇难者"是整个故事的核心隐喻。

### 场景 20: 沉没学校 - 残暴者的阻挡

**分镜**:
1. **建立镜头**：一所废弃小学，操场积水，教学楼半淹没
2. **内景 - 走廊**：墙上贴满儿童画，内容是"爸爸去上班""妈妈在做饭"
3. **特写**：其中一幅画的角落写着"小雨 画"

**对白**:
```
小雨（第一次开口，声音微弱）：
"这是……我的学校……"

沈迴（蹲下）：
"你记得吗？"

小雨（摇头）：
"不记得了……但这里……很熟悉……"
```

**心理意义**:  
小雨开始"说话"，象征她从**创伤的麻木**中苏醒。这是因为沈迴归还了她父亲的名字，给了她哀悼的权利。

---

### 场景 21: 残暴者遭遇战

**分镜**:
1. **走廊尽头**：一个巨大的身影挡住去路——残暴者
2. **外形**：重装甲般的躯体，移动缓慢，面部是一张扭曲的"拒绝"表情
3. **它开口**（低沉、缓慢）：
   ```
   "不……承认……
   不是……我的错……
   不……不……不……"
   ```

**对白**:
```
沈迴（后退）：
"它在说什么？"

老吴（严肃）：
"它在拒绝。拒绝承认，拒绝面对。
这东西……正面打不过的。"

沈迴（理解）：
"所以要绕路。"
```

**战斗机制**:
- 残暴者正面无法击退（即使用十字架也只能短暂定身）
- 玩家必须利用教室、桌椅绕到它背后
- 背后攻击有效，会让它失衡

**象征意义**:  
残暴者代表**心理防御机制中的"否认"**。你无法正面击破一个拒绝承认的人，你只能绕过它，或者等它自己放下防御。

**分镜细节**:
- 特写：残暴者的"脸"上有泪痕（它也在痛苦，只是不承认）
- 击败后，它不会消失，而是跪倒在地，发出低沉的哭泣声

---

### 场景 22: 发现小雨的秘密

**分镜**:
1. **教室内**：小雨走到一张课桌前，停下
2. **特写**：课桌上刻着"王小雨"三个字
3. **小雨打开抽屉**：里面有一个日记本

**日记内容**（部分）:
```
9月15日 晴
今天爸爸说要加班，让我在门口等他。
我等了好久好久，天都黑了。

9月16日 （没有天气）
爸爸还没来。
妈妈说爸爸出差了。
但我觉得……爸爸是不是不要我了？

9月17日 （手写变得凌乱）
我要去找爸爸。
妈妈不让我去，但我要去。
疗养院……爸爸在那里……

（最后一页空白，只有一滩水渍）
```

**对白**:
```
小雨（哭泣）：
"我去找他了……但是……雾……
雾把我……"

沈迴（抱住她）：
"你没做错。你只是想找爸爸。"

小雨（抽泣）：
"沈医生……爸爸还会回来吗？"

沈迴（沉默片刻，温和但坚定）：
"他不会回来了。但他一直爱着你。
这点，雾记得，我也记得。"
```

**心理刻画**:  
这是小雨的**情感释放**时刻。她终于明白父亲不会回来，但她也得到了确认：父亲的死不是因为不爱她。这是**哀悼的完成**。

---

### 场景 23: 第三幕结束 - 教堂的承诺

**分镜**:
1. **返回教堂**：沈迴、老吴、小雨三人回到祭坛前
2. **特写**：沈迴看着剩余的灰烬罐（还有数十个）
3. **决心**：他开始一个个拾取

**对白**:
```
老吴：
"你要全部归还？"

沈迴：
"我欠他们的。"

小雨（牵住沈迴的手）：
"我帮你。"

沈迴（微笑，眼中有泪）：
"谢谢。"
```

**蒙太奇**:
- 镜头：沈迴一个个归还灰烬
- 每个灰烬化作光点时，闪过一段记忆碎片
- 有些是呼救，有些是道别，有些是"对不起"
- 画面叠化：名字从模糊变清晰

**过场文字**:
```
"你记起了他们的名字。
你归还了他们的存在。
但真相……还在更深处。"

第四层 - 破晓海岸
```

**心理状态转变**:  
第三幕结束时，沈迴从**抑郁**进入**接受**的阶段。他不再问"我是否做错了"，而是问"我现在能做什么"。

**画面定格**:
- 最后一个灰烬归还时，镜头推远
- 教堂被温暖的金光包围
- 三个人的剪影站在光中
- 淡出

---

## 第三幕主题总结

**核心主题**: 归还与哀悼

**情感弧**:
- 沈迴：从"我无法原谅自己"到"但我可以承担责任"
- 老吴：从旁观到参与，开始直面自己的不作为
- 小雨：从创伤的麻木到完成哀悼，获得继续生活的力量

**象征体系**:
- 灰烬 = 被遗忘的名字
- 归还仪式 = 让存在被记住
- 残暴者 = 心理防御（否认）
- 教堂 = 神圣的责任空间

**视觉风格**:
- 从前两幕的冷色调（灰、蓝、锈红）
- 转向暖色调（金色、淡黄、晨曦）
- 雾开始变薄，但仍未完全消散

**音效设计**:
- 减少恐怖音效
- 加入哀伤但温暖的弦乐
- 远处传来海浪声（暗示接近海岸=出口）
